17 /** A 2D field of double distributed over a Cartesian grid.
18  *
19  * A Distributed2DField represents a 2D field of double distributed in blocks
20  * over a Cartesian grid described by the CartesianDistribution2D exposed by
21  * the distribution() function.
22  * Each local block features ghost zones that can be synchronized using the
23  * sync_ghosts() function.
24  *
25  * The field offer multiple "view":
26  * - full_view() is a coordinate system that includes the ghosts, the point at
27  * coordinate (0,0) is in the bottom left of the ghost corner and the
28  * View::extents() includes all ghosts,
29  * - noghost_view() is a coordinate system that excludes the ghosts, the point
30  * at coordinate (0,0) is in the bottom left corner of the local block and the
31  * View::extents() excludes all ghosts,
32  * - ghost_view() offers a view inside each of the ghost: LEFT, RIGHT, UP and
33  * DOWN.
34  *
35  * The distance between two grid points can be accessed through the delta()
36  * function and the time-coordinate of the field can be set and accessed using
37  * the time() function.
38  */
