9 /** Description of a 2D Cartesian data distribution.
10  *
11  * A CartesianDistribution2D represents the organization of a set of processes
12  * on a Cartesian grid. Each process has a coordinate in this grid that can be
13  * accessed with the coord() function. The total number of process in each
14  * dimension is provided by the extents() function.
15  *
16  * The underlying MPI communicator is provided by the communicator() function
17  * and its size as well as the rank of the local process are available through
18  * the size() and rank() functions.
19  *
20  * The rank of the neighbouring processes can be accessed with the
